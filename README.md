# @uni-helper/uni-env

> 检测当前 uni-app 的环境变量

## 安装

```sh
# Using Yarn
yarn add @uni-helper/uni-env

# Using npm
npm i @uni-helper/uni-env

# Using pnpm
pnpm i @uni-helper/uni-env
```

## 使用

```js
// ESM
import { isMpWeixin } from "@uni-helper/uni-env";

// CommonJS
const { isMp } = require("@uni-helper/uni-env");
```

你可以从 [./src/index.ts](./src/index.ts) 中查看更多标志以及如何判断的。
